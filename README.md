# 🌡️ Conversor de Temperatura em C

Projeto desenvolvido para praticar lógica de programação utilizando a linguagem C, implementando conversões entre diferentes escalas de temperatura por meio de funções, estruturas condicionais e menus interativos.

## 📋 Funcionalidades

- ✅ Conversão de Celsius para Fahrenheit
- ✅ Conversão de Fahrenheit para Celsius
- ✅ Conversão de Celsius para Kelvin
- ✅ Conversão de Kelvin para Celsius
- ✅ Menu interativo
- ✅ Validação de temperaturas inválidas em Kelvin

## 🛠️ Tecnologias

- Linguagem C

## 📚 Conceitos utilizados

- Variáveis
- Entrada e saída de dados (`scanf` e `printf`)
- Estruturas condicionais (`if` e `switch`)
- Laço de repetição (`do...while`)
- Funções
- Operações matemáticas

## ▶️ Como executar

```bash
gcc src/main.c -o conversor
./conversor
```

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar a implementação de funções, organização de código e estruturas de controle, aplicando fórmulas de conversão entre diferentes escalas de temperatura.
